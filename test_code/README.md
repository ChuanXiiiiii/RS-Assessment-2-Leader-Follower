# Test Code

This folder contains all test code and scripts for the leader-follower robot project.

## Contents

- `bumpsensor_rc_test.ino`: Bump sensor RC timing data logger.
- `final_follower.ino`: Follower robot implementation with dual PID control.
- `leader_3modeversion.ino`: Leader robot with multi-mode movement.
- `linesensor_rc_test.ino`: Line sensor RC timing data logger.
- `stable_version.ino`: Stable version of the follower robot.

## Usage

Refer to individual script documentation for usage instructions.
