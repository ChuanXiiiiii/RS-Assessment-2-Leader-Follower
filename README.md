## Folders Description

### `test_code/`
This folder contains all test code and scripts developed during the project, including:
- `bumpsensor_rc_test.ino`: Bump sensor RC timing data logger.
- `final_follower.ino`: Follower robot implementation with dual PID control.
- `leader_3modeversion.ino`: Leader robot with multi-mode movement.
- `linesensor_rc_test.ino`: Line sensor RC timing data logger.
- `stable_version.ino`: Stable version of the follower robot.

### `videos/`
This folder contains intermediate robot performance demonstration videos, documenting:
- `extreme-angle tracking.mp4`: Demonstration of tracking at extreme angles.
- `extreme-range tracking.mp4`: Demonstration of tracking at extreme ranges.

## Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/ChuanXiiiiii/RS-Assessment-2-Leader-Follower.git
   ```

2. Navigate to the test code folder to explore the implementation:
   ```bash
   cd RS-Assessment-2-Leader-Follower/test_code
   ```

3. Check the videos folder for demonstration recordings.
